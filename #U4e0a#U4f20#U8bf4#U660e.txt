唐人日报后台单独修复补丁

只包含 admin 文件夹，不包含根目录 index.html，不会覆盖新闻前台首页。

GitHub 上传方法：
1. 解压本压缩包。
2. 打开 huarenusvisa/trrb 仓库 main 分支。
3. 将 admin 文件夹上传到仓库根目录。
4. 最终结构必须是：
   /index.html          前台新闻首页
   /admin/index.html    后台登录页
   /admin/admin.js
   /admin/styles.css
5. 提交后，在 Netlify 解锁自动发布，等待 Published。
6. 测试：https://new.trrb.net/admin/

不要把 admin/index.html 上传成仓库根目录的 index.html。
