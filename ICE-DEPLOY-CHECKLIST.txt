唐人日报 ICE 自动发布部署清单

[ ] 1. 将压缩包内容上传到唐人日报 GitHub 仓库根目录
[ ] 2. GitHub Secrets 添加 X_BEARER_TOKEN
[ ] 3. GitHub Secrets 添加 OPENAI_API_KEY
[ ] 4. GitHub Secrets 添加 OPENAI_MODEL（建议 gpt-4.1-mini）
[ ] 5. Settings → Actions → General → Workflow permissions 选择 Read and write
[ ] 6. Actions → ICE执法自动抓取发布 → Run workflow
[ ] 7. 打开 https://new.trrb.net/topic/ice/
[ ] 8. 检查首页“ICE执法”卡片是否显示“自动更新”
[ ] 9. 检查 data/ice-pending.json 是否有未通过校验的内容
[ ] 10. 正式域名切换后，把 SITE_URL 改为 https://trrb.net
