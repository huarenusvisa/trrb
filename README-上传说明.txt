唐人日报文章详情页修复包

修复内容：恢复缺失的 article.js，解决 article.html 打开后文章内容无法加载、控制台 404 的问题。

上传方法：
1. 解压本压缩包。
2. 将 article.js 上传到 GitHub 仓库根目录。
3. 确保它与 article.html、index.html、styles.css 位于同一级目录。
4. 提交到 main 分支并等待 Netlify 自动部署。
5. 部署完成后打开：https://trrb.net/article.js?v=29.7
6. 如果能看到 JavaScript 代码，说明文件已恢复。
7. 再用 Ctrl+Shift+R 强制刷新文章页面。
